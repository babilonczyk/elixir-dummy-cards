# elixir-dummy-cards

### Simple app that tests basic functionalities of elixir like: creating modules, methods, pattern matching, working on lists, writing doc files and documentation.

## To run it:

Install dependencies:
`mid deps.get`

Run interactive elixir:
`iex -S mix`

Run tests: 
`mix test`

Generate docs:
`mix docs`
`cd docs`
`open index.html`
